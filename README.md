# directory-tools
A lightweight Python toolset for directory/file statistics and search
# Directory Tools
一个轻量级的Python工具集，用于目录/文件的大小计算、数量统计、内容搜索、路径匹配。

## 功能列表
- `dir_size.py`：递归计算目录总大小，支持字节/KB/MB/GB格式化输出；
- `dir_stats.py`：统计目录下文件总数、子目录总数（递归）；
- `file_search.py`：在指定文件中查找目标字符串，输出行号和匹配内容；
- `pattern_search.py`：递归查找目录下匹配指定模式的文件路径。

## 环境要求
- Python 3.6+
